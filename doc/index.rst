Welcome to Neuronmi's documentation!
====================================

Neuronmi is a Python package to simulate neuronal activity with finite element methods with a
high-level and accessible API.

With :code:`neuronmi` one can:

- generate 3D meshes with neurons and extracellular probes
- run simulations with different kind of solvers
